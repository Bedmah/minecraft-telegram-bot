# minecraft-telegram-bot

Telegram-бот для управления Minecraft-сервером через RCON.  
Поддерживает привязку Telegram-аккаунта к Minecraft нику без плагинов, управление игроком через кнопки и админ-панель.

## Возможности

### Для игроков
- Просмотр онлайна и списка игроков
- Телепорт:
  - к игроку онлайн (через кнопки)
  - по координатам (x y z)
- Смена режима игры (gamemode) только для себя
- Привязка и отвязка аккаунта Minecraft

### Привязка аккаунта без плагинов
Используется vanilla-механика:
- `/scoreboard trigger`
- `/trigger`

Процесс:
1. Бот выдает одноразовый код
2. Игрок вводит код в Minecraft через `/trigger`
3. Бот проверяет значение через RCON и связывает аккаунты

### Для админа
- Админ-панель в Telegram (нижняя панель кнопок)
- Просмотр онлайна
- Запуск сервера через `start.sh`
- Стоп и рестарт сервера
- Создание бэкапа (через плагин Backuper)
- Просмотр логов запуска
- Список пользователей, использующих бота
- Выполнение произвольных RCON-команд

## Требования

- Linux сервер (Ubuntu, Debian и аналоги)
- Python 3.10 или новее
- Minecraft сервер (Paper или Spigot)
- Включенный RCON
- Доступ бота к RCON порту

Опционально:
- `start.sh` для запуска сервера из бота
- Плагин Backuper для бэкапов
