# minecraft-telegram-bot

Telegram-бот для управления Minecraft-сервером через RCON.  
Поддерживает привязку Telegram-аккаунта к Minecraft нику **без плагинов**, управление игроком через кнопки и админ-панель.

Проект ориентирован на небольшие сервера и игру с друзьями.

---

## Возможности

### Для игроков
- Просмотр текущего онлайна и списка игроков
- Телепорт:
  - к игроку онлайн (через кнопки)
  - по координатам (x y z)
- Смена режима игры (gamemode) **только для себя**
- Привязка и отвязка аккаунта Minecraft

### Привязка аккаунта без плагинов
Используется vanilla-механика Minecraft:
- `scoreboard trigger`
- команда `/trigger`

Процесс:
1. Бот выдает одноразовый код
2. Игрок вводит код в Minecraft через `/trigger`
3. Бот проверяет значение через RCON и связывает Telegram и Minecraft аккаунты

### Для администратора
- Админ-панель в Telegram (нижняя панель кнопок)
- Просмотр онлайна
- Запуск сервера через `start.sh`
- Остановка и рестарт сервера
- Создание бэкапа мира (через плагин Backuper)
- Просмотр логов запуска
- Просмотр списка пользователей, использующих бота
- Выполнение произвольных RCON-команд

---

## Требования

- Linux сервер (Ubuntu, Debian и аналоги)
- **Python 3.10 или новее**
- Minecraft сервер (Paper или Spigot)
- Включённый RCON
- Доступ бота к RCON порту

Опционально:
- `start.sh` для запуска сервера из бота
- Плагин Backuper для бэкапов

## Установка

# 1. Проверка Python
python3 --version

# 2. Установка Python и pip (если нужно)
sudo apt update
sudo apt install -y python3 python3-pip

# 3. Установка зависимостей
pip3 install -r requirements.txt

# (если requirements.txt нет)
pip3 install python-telegram-bot mcrcon

# 5. Создание конфига
cp .env.example .env
nano .env

# 6. (Опционально) сделать скрипт запуска исполняемым
chmod +x bot.sh

# 7. Запуск бота
python3 bot.py

# ИЛИ запуск в screen
screen -dmS mc-bot ./bot.sh
