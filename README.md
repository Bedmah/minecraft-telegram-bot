# minecraft-telegram-bot

Telegram-бот для управления Minecraft-сервером через RCON.
Позволяет привязать Telegram-аккаунт к Minecraft нику без использования плагинов, управлять игровыми действиями через кнопки и админ-панель прямо в Telegram.

Проект рассчитан на небольшие серверы и приватную игру с друзьями.

================================
ОСНОВНЫЕ ВОЗМОЖНОСТИ
================================

ДЛЯ ИГРОКОВ
- Просмотр текущего онлайна и списка игроков
- Телепортация:
  - к игроку, находящемуся онлайн (через кнопки)
  - по координатам (x y z)
- Смена режима игры (gamemode) только для себя
- Привязка и отвязка Minecraft-аккаунта

--------------------------------
ПРИВЯЗКА АККАУНТА БЕЗ ПЛАГИНОВ
--------------------------------

Используется стандартная vanilla-механика Minecraft:
- scoreboard trigger
- команда /trigger

Процесс:
1. Бот выдает игроку одноразовый код
2. Игрок вводит код в игре через /trigger
3. Бот проверяет значение через RCON и связывает Telegram и Minecraft аккаунты

--------------------------------
ВОЗМОЖНОСТИ АДМИНИСТРАТОРА
--------------------------------

- Админ-панель в Telegram
- Просмотр онлайна сервера
- Запуск сервера через start.sh
- Остановка и рестарт сервера
- Создание бэкапа мира (через плагин Backuper)
- Просмотр логов запуска
- Просмотр списка пользователей бота
- Выполнение произвольных RCON-команд

--------------------------------
ТРЕБОВАНИЯ
--------------------------------

Обязательно:
- Linux сервер (Ubuntu, Debian и аналоги)
- Python 3.10 или новее
- Minecraft сервер (Paper или Spigot)
- Включенный RCON
- Доступ к RCON-порту

Опционально:
- start.sh для запуска сервера из бота
- Плагин Backuper для бэкапов

--------------------------------
УСТАНОВКА
--------------------------------

1. Проверка версии Python
python3 --version

2. Установка Python и pip (если требуется)
sudo apt update
sudo apt install -y python3 python3-pip

3. Установка зависимостей
pip3 install -r requirements.txt

Если файла requirements.txt нет:
pip3 install python-telegram-bot mcrcon

4. Создание конфига
cp .env.example .env
nano .env

5. (Опционально) Сделать скрипт запуска исполняемым
chmod +x bot.sh

6. Запуск бота
python3 bot.py

Или запуск в screen:
screen -dmS mc-bot ./bot.sh
